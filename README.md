# Manthedan Tap

## How do I install these formulae?

`brew install manthedan/tap/<formula>`

Or `brew tap manthedan/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "manthedan/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
